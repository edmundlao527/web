高媽媽餐館

一間位於輔仁大學隔壁，工商城裡的一間便當店，吸引大量輔大學生每天前往光顧，是一間非常具人氣的店。

高媽媽飯館在514巷已佇立接近20年之久，這裡賣的其實是一般便當店常見的雞腿飯、叉燒飯、排骨飯、雞排飯...等等，價位從60元到80元不等，相當有能夠吸引學生的魅力。

輔大周邊便宜又好吃的代表之一，高媽媽飯館讓超多輔大人畢業後念念不忘，不僅是因為東西好吃又便宜，老闆的親切感也是很多輔大人心中的小小撫慰。從鮭魚、鱈魚飯，到雞腿、獅子頭、雞排飯等，讓男生輕鬆吃飽的份量加上每天不同的豐富配菜，內用還可以免費加飯喝紅茶。
